# Workday Learning & Certification Tracker Agent

## Purpose
Automate tracking and reminders for Workday trainings and certifications using Outlook email notifications as the data source, ensuring timely completion and reducing manual follow-ups.

## Architecture Overview

### Tooling
- **Power Automate**: Core automation engine
- **Outlook Mail**: Data source and notification delivery
- **Microsoft Teams**: Notification delivery
- **Outlook Calendar**: Availability integration

### Data Source (Revised)
- Training and certification details are **extracted from Outlook emails generated by Workday**
- No direct API or connector integration with Workday
- Power Automate parses:
  - Training/certification names
  - Due dates or expiry dates
  - Action links (Workday, MyTime & Expense, etc.)

## Core Functionality

### Email Monitoring
- Monitor incoming Outlook mails from Workday (or configured senders)
- Extract and structure learning data from email content
- Track status: **Completed / Pending / Due Soon / Overdue**
- Maintain user-specific learning overview

### Reminder Logic
- **Daily reminders** for pending and due-soon items
- **Weekly summary** with overall completion status
- **Escalation alerts** at T-3 and T-1 days before deadline
- Suppresses reminders once completion is confirmed or no new alerts arrive

### User Interaction
- Notifications delivered via **Teams** and **Outlook**
- Reminder emails include extracted **action links** for quick access

### Calendar Integration
- Reads Outlook availability

## Security & Compliance
- Role-based access
- Enterprise authentication
- Email-driven, read-only data collection (no Workday write-back)

## Outcome
- No dependency on Workday APIs
- Improved compliance using existing email notifications
- Reduced HR and manager manual tracking effort
- Centralized, automated learning reminders

## Implementation Options

### Option 1: Simple Email-Only Approach (15 minutes)
**Best for:** Small teams, quick setup, minimal maintenance
- Single flow: Email arrives → Parse → Send reminder
- No storage required
- No daily reminders or tracking
- See `SIMPLE_EMAIL_ONLY.md` for setup

### Option 2: Full Solution with Storage (90 minutes)
**Best for:** Organizations, comprehensive tracking, daily reminders
- Multiple flows with SharePoint/Excel storage
- Daily reminders, escalation alerts, weekly summaries
- Complete tracking and reporting
- See `IMPORT_GUIDE.md` for setup
